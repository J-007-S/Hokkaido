九州10日攻略

福冈→久留米→别府→由布院→阿苏→熊本→鹿儿岛→长崎

九州详细行程

11/19（周二）
抵达福冈（晚）

抵达福冈机场

休息或到博多站/天神短逛（拉面、屋台）。

住宿：福冈（博多/天神）

11/20（周三）
福冈市区（用一日通票）→ 晚上前往久留米住

市内一日通票游福冈

大濠公园 / 福冈城遗址

太宰府天满宫（Dazaifu）（乘西铁或JR+巴士约 30–50 分）(枫叶)

乘列车/快速（普通或快速列车，使用 IC 卡或单程票）前往久留米

11/21（周四）
【激活 JR 7 日 Pass】久留米 → 别府（Beppu）/由布院方向，入住（别府或由布院）

久留米 → 大分/别府 方向

别府地狱巡游（Beppu Jigoku）（几个地狱温泉点可坐地面巴士），傍晚可去温泉旅馆泡汤。

11/22（周五）
别府 → 由布院（Yufuin）半日游 → 傍晚返回久留米（住久留米）

从别府出发前往 由布院（短途特急或普通列车，约 30–60 分钟）

逛 金鳞湖、由布院散步小街、艺术馆与温泉咖啡店（适合慢慢走、拍照、泡脚）

下午/傍晚：由布院 → 久留米 返回（使用 JR Pass）

晚上：入住 久留米

11/23（周六）
久留米 → 阿苏（Aso）一日游 → 熊本市区短停 → 返回久留米（住久留米）

久留米出发 → 前往阿苏地区

阿苏：参观 阿苏火山观景点 / 草千里（Kusasenri）

从阿苏前往 熊本市区（车程约 1 小时左右），参观 熊本城（外观/周边）、城下町短逛、尝熊本拉面。

熊本 → 久留米

11/24（周日）
久留米 → 鹿儿岛（Kagoshima）

久留米出发 → 乘新干线/特急向南到 鹿儿岛中央站（JR Pass 可覆盖新干线区间的普通列车/部分新干线需要确认）

乘渡轮去 樱岛（Sakurajima） 近观火山海景、访问城山公园展望台

住宿：鹿儿岛市内

11/25（周一）
鹿儿岛周边自然（樱岛 / 雾岛 / 指宿任选） → 傍晚返回久留米（住久留米）

选项 A（偏火山/山景）：雾岛（Kirishima）一日游（火山、喷泉、山间温泉）

选项 B（海景/体验）：指宿（Ibusuki）砂蒸温泉（体验独特的砂浴）

选项 C（轻松）：鹿儿岛市内多处自然点 + 温泉休闲

从鹿儿岛返回久留米

11/26（周二）
久留米周边放松日/备选 → 久留米住宿

选项 1：柳川（Yanagawa） 乘 川下り（掘割船），赏运河风光与河畔茶室，约半天行程

选项 2：佐贺一日游，随后回久留米。

住宿：久留米

11/27（周三）
久留米 → 长崎（Nagasaki）

早上：久留米出发 → 长崎

下午：抵达长崎后：平和公园 / 野母崎海岸线 / 稻佐山展望台（傍晚上稻佐山看城市/港湾夜景，非常推荐）

住宿：长崎市内（1 晚）

说明：此段也在 JR Pass 有效期内（11/21–11/27），当天回程或转车均可用通票。

11/28（周四）
长崎（早上自由） → 中午离境（长崎机场 / 长崎出境）

早上：若时间充裕可再去 出岛或滨海散步

中午：前往长崎机场

北海道详细行程

11/21（周四）
抵达函馆（晚）

抵达函馆机场，乘坐机场巴士前往函馆市区（约30分钟，票价1200日元）

住宿：函馆站附近（方便后续行程赶车）

晚餐：函馆站周边海鲜居酒屋，推荐海鲜丼、烤扇贝

11/22（周五）
函馆市区一日游 → 傍晚登稻佐山看夜景

上午：函馆朝市（品尝新鲜海鲜、现切金枪鱼）→ 函馆红砖仓库群（复古建筑拍照、文创店购物）

中午：朝市附近海鲜定食午餐

下午：函馆植物园（冬季温室花卉）→ 函馆圣约翰教堂（欧式建筑）

傍晚：乘坐缆车登稻佐山（往返票价1500日元），观赏世界三大夜景之一的函馆夜景

交通：市内巴士（一日券800日元，覆盖主要景点）

景点 开放时间 门票
函馆朝市 5:00-14:00 免费（消费自理）
红砖仓库群 10:00-19:00 免费
稻佐山缆车 10:00-22:00（冬季） 往返1500日元
11/23（周六）
函馆 → 登别（温泉乡体验）

上午：函馆站乘坐JR特急「北斗号」前往登别（约2小时，JR Pass可用）

中午：登别温泉街午餐，推荐温泉蛋定食、烤羊肉

下午：登别地狱谷（火山地貌、硫磺泉眼）→ 大汤沼（雾气缭绕的温泉湖）→ 登别熊牧场（喂食熊群）

晚上：温泉旅馆泡露天风吕（登别温泉为北海道三大温泉之一，泉质含硫磺，美容养颜）

住宿：登别温泉旅馆（推荐含早晚餐的和式房型）

登别温泉注意事项：

  • 泡汤前需先淋浴洗净身体，毛巾不可入池
  • 酒后、饭后1小时内不可泡汤
  • 露天风吕冬季气温较低，建议披浴袍往返
11/24（周日）
登别 → 札幌市区（购物+美食）

上午：登别站乘坐JR特急前往札幌（约1.5小时，JR Pass可用）

中午：札幌站附近午餐，推荐札幌味噌拉面

下午：大通公园（冬季可能有冰雪节预热）→ 北海道厅旧本厅舍（红砖建筑，免费参观）→ 狸小路商店街（购物、药妆、特产）

晚上：薄野欢乐街晚餐+散步（居酒屋、海鲜烧烤）

住宿：札幌市中心（薄野或大通公园附近）

狸小路必买清单

  • 白色恋人巧克力（限定口味）
  • Royce'生巧克力（冷藏包装）
  • 北海道牛奶糖、果干
  • 药妆（资生堂、珂润等）
11/25（周一）
札幌 → 小樽一日游（复古运河+美食）

上午：札幌站乘坐JR函馆本线前往小樽（约30分钟，JR Pass可用），推荐08:00前班次（避开旅行团）

行程安排：

  • 09:00-10:30 南小樽站→小樽运河（步行15分钟，清晨拍照人少）
  • 10:30-12:00 小樽运河+运河仓库街（可乘运河游船，30分钟/1500日元）
  • 12:00-13:30 午餐：运河附近「小樽海鲜食堂」，推荐海鲜丼、烤扇贝
  • 13:30-15:00 小樽音乐盒堂+北一硝子馆（复古音乐盒、手工玻璃制品）
  • 15:00-16:00 小樽童话十字路口+白色恋人巧克力工坊（小樽店）

下午：16:30前乘坐JR返回札幌（约35分钟，班次密集）

小樽美食推荐：

LeTAO芝士蛋糕 双层芝士口感绵密，限定口味必尝
烤鱿鱼干 运河边现烤，咸香有嚼劲
海鲜丼 新鲜北海道海鲜，盖饭量大料足
11/26（周二）
札幌 → 旭川（动物园萌宠+拉面）

上午：札幌站乘坐JR特急「超级神威号」前往旭川（约1小时40分钟，JR Pass可用），推荐08:00出发→09:40抵达

下午：旭川动物园游玩（9:00-16:30开放，门票800日元）

核心看点：

  • 北极熊（室内观赏通道+户外雪地活动）
  • 企鹅（11月底可能开启冬季「企鹅散步」预热）
  • 雪豹、海豹、猫头鹰

交通：旭川站→旭川动物园（巴士20分钟，票价310日元；出租车15分钟，2500日元）

晚上：旭川站附近「旭川拉面村」晚餐，推荐梅光轩（酱油拉面）、青叶（味噌拉面）

住宿：旭川市内

11/27（周三）
旭川 → 美瑛（田园雪景）→ 札幌

上午：旭川站乘坐JR普通列车前往美瑛（约20分钟，JR Pass可用），推荐08:30出发→08:55抵达

美瑛游玩方式（冬季推荐包车游，4-5人同行约15000-18000日元/天）

核心景点（超广角之路+拼布之路）：

  • 圣诞树（超广角之路标志性景点，雪地孤树拍照出片）
  • 七星之树（曾作为七星香烟广告背景）
  • 亲子树（两棵相依的树木，雪地中温馨）
  • 拼布之路（初雪覆盖后像白色拼布）

午餐：美瑛站附近「美瑛乡村料理 北野」，推荐烤羊肉定食、蔬菜天妇罗

下午：16:30前乘坐JR特急返回札幌（约1小时30分钟）

冬季美瑛注意事项：

  • 气温-5~0℃，需穿戴防风羽绒服、雪地靴、厚手套、暖宝宝
  • 雪地路面滑，步行/骑行需放慢速度
  • 手机低温耗电快，携带充电宝
11/28（周四）
札幌 → 新千岁机场 → 返程

上午：札幌站地下「札幌一番街」购买伴手礼（白色恋人、Royce'、六花亭等）

交通：札幌站乘坐JR快速机场线前往新千岁机场（约45分钟，JR Pass可用，每15分钟一班）

⚠️ 提示：国际航班提前3小时抵达机场，国内航班提前2小时

机场补充购物：新千岁机场免税店（美妆、酒类、零食）
